# Homebrew Tap for Spore CLI

This is the official [Homebrew](https://brew.sh) tap for the Spore CLI.

## Installation

```bash
brew tap sporeintel/tap
brew install spore
```

## Updates

```bash
brew upgrade spore
```

The formula is automatically updated when new Spore CLI releases are published.

## About Spore

Spore is a distributed AI inference network. Learn more at [sporeintel.com](https://sporeintel.com).
